# KKN-WaterMonitoring
Proyek KKN PPM UGM 2021 di Desa Nanggulan. <br>
**Sistem monitoring kualitas air kolam berbasis IoT**. <br>
Menggunakan <br> 
1. Raspberry Pi<br> 
2. Node-RED, <br>
3. ESP32 <br>

<br><br>
Sistem Operasi : Raspbian OS
